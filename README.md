**Posso com a linguagem criar listas**

**Lista numerada:**
1. Número um
   1. Subitem do Número um
2. Número dois
4. Número três

**Lista demarcada:**
* Número um (feito com asterisco)
- Número dois (feito com traço)
   - Subitem do número dois  (feito com traço)
* Número três (feito com asterisco)
